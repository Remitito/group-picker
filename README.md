**Group Picker**

This is a tool for assigning groups which improves upon existing options according to teachers' feedback.

**How to run**

To run it locally: 1) Clone the repo 2) Enter 'npm install' in the terminal 3) Enter 'npm start' in the terminal 4) Go to http://localhost:3000/ in the browser

**Live Version**

The live version can be found at: https://remitito.github.io/group-picker/

**Features which are not found in existing platforms**

- Press the hand icon to enter the "Avoid List", where you can choose who that person should not be grouped with
- You can use the arrows to switch students directly or move them to the next or previous group without restarting the process

**Tech Stack**

- This is tool was built using React
